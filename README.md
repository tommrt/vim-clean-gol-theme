# vim-clean-gol-theme
A clean green orange light (gol) colorscheme for vim

### Main Colors
- #18874b (dark green)
- #F68D2D (orange)
- #7eba48 (light green)

- [x] Markdown

- [ ] Python
- [ ] Rust
